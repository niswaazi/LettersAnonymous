Anonymous Message Web adalah aplikasi berbasis Laravel yang memungkinkan pengguna mengirim pesan atau surat secara anonim melalui konsep mading digital interaktif. Pengguna dapat menulis pesan kepada seseorang tanpa menampilkan identitas pengirim, sehingga komunikasi menjadi lebih bebas, nyaman, dan menarik.

Aplikasi ini dibangun menggunakan framework Laravel dengan arsitektur MVC (Model–View–Controller) dan menggunakan MySQL sebagai database untuk menyimpan data pesan, pengguna, dan informasi pendukung sistem.

## Fitur Utama

* Mengirim pesan anonim
* Menampilkan daftar pesan pada mading digital
* Pencarian pesan berdasarkan nama penerima
* Fitur PIN untuk mengunci dan membuka pesan tertentu
* Dashboard admin untuk mengelola pesan
* Validasi input pengguna
* Proteksi anti-spam
* Penyimpanan data menggunakan database MySQL

## Teknologi yang Digunakan

* Laravel 12
* PHP 8
* MySQL
* phpMyAdmin
* HTML
* CSS
* JavaScript
* Blade Template Engine

## Tujuan Proyek

Proyek ini dibuat sebagai media komunikasi anonim berbasis web sekaligus sebagai implementasi pembelajaran pengembangan aplikasi menggunakan Laravel dan pengelolaan database MySQL.
